# 🎯 1. نظرة عامة عن اللعبة

**ملك الصحراء** لعبة استراتيجية صحراوية متعددة اللاعبين (RTS) تعمل بالكامل في المتصفح — جوال وديسكتوب.

**الفكرة:** تبدأ كمسافر ضائع في واحة صغيرة، وتبني مملكتك عبر 5 فصول قصصية حتى تصبح ملك الصحراء.

**الميزات الأساسية:**
- 🎮 PvP مباشر + PvE + Battle Royale
- 📖 قصة تفاعلية بخيارات ومكافآت (5 فصول، 25 مشهد)
- 🏗️ بناء وترقية المباني في 5 قرى
- 🗡️ 6 أسلحة قابلة للترقية (5 نجوم)
- 👑 نظام تحالف + غارات
- 🌴 واحات للسيطرة — دخل سلبي
- 🔄 Prestige (إعادة الميلاد) — 5 مستويات
- 🎊 أحداث عشوائية + مهام يومية + إنجازات
- 🛒 متجر + تصنيع + مخزون

# 🎮 4. حلقة اللعب — Gameplay Loop

## 4.1 الدقيقة الأولى — دخول اللاعب الجديد

```
1. فتح اللعبة → شاشة تحميل بعجلة دوارة + شريط تقدم
2. إدخال اسم المستخدم → التحقق من قاعدة البيانات
3. تظهر القصة — الفصل 1، مشهد "الوصول إلى الواحة"
   → اللاعب يختار: 🔨 ابدأ فوراً في بناء الخيمة أو 🔭 استكشف المنطقة
   → يحصل على مكافأة فورية
4. بعد القصة → التدريب (14 خطوة)
   → "هذا شريط الموارد"
   → "شاشة الترقية — ابنِ مباني الواحة"
   → "المتجر — اشترِ وبيع"
   → ... إلخ
5. يصل إلى الشاشة الرئيسية → "الأراضي" 🗺️
   → 5 مباني 🏗️ كلها 🔒 مقفلة
```

## 4.2 اليوم العادي — ماذا يفعل اللاعب؟

### كل دقيقة:
```
1. 🗺️ ينظر إلى الخريطة → يرى لاعبين آخرين يتحركون
2. 🏗️ يضغط على مبنى مقفول → ⚔️ يقاتل وحشاً → يبني المبنى
3. 📈 يرقي المباني للحصول على دخل سلبي
4. ⚔️ يذهب لشاشة الحرب → يدخل المغامرة → يقتل وحوش
5. 👊 يهاجم لاعبين آخرين (PvP) → يكسب فلوس
```

### كل يوم:
```
1. 📅 يستلم المكافأة اليومية (7 أيام)
2. 📋 يفتح المهام → يكمل المهام اليومية (3 مهام)
3. 📖 يتقدم في القصة → يفتح قرية جديدة
4. 🗡️ يطور أسلحته (شراء + ترقية نجوم)
5. ⭐ يرقّي جنوده وتدريبهم
6. 🔬 يبحث في شجرة البحوث
7. 🎊 يتابع الأحداث النشطة
8. 🌴 يسيطر على واحات جديدة
```

### كل أسبوع:
```
1. 🔄 إعادة الميلاد (Prestige) إذا وصل المستوى 110
2. 👑 غارات التحالف — قتال زعماء مع أعضاء التحالف
3. 🏆 منافسة على لوحة المتصدرين
4. 🎁 7 أيام مكافآت ← مكافأة الأسبوع
```

## 6.2 معادلة القوة

```
قوة الجيش = unitLevel × trainingLevel × 10
           + مستوى الأسلحة (مجموع نجوم × 50)
           + مستوى البطل × 20
           + مستوى التحالف × 100
           + بونص المعرفة
           + مضاعف Prestige
```

## 6.3 PvP — خسارة

عند الخسارة في PvP:
- ❌ تفقد 5% من رصيدك الكلي 💵
- ❌ تفقد 10% من قوتك (min 500)
- 🛡️ حماية لمدة 30 ثانية — لا يمكن مهاجمتك
- 📱 تظهر شاشة موت مع عد تنازلي 10 ثوانٍ
- 🔄 زر "العودة للخريطة" أو الانتظار للعودة التلقائية

| **مدني (civic)** | مقر القائد، خيمة، عرش |

```
3. ✅ فتح المبنى ← دفع تكاليف البناء
4. 🏗️ المبنى شغال ← يدر دخلاً
5. 📈 ترقية ← دفع تكاليف ← إنتاج أعلى
```
