## 🎯 ما هو المشروع؟

لعبة استراتيجية متعددة اللاعبين (ملك الصحراء) — PWA كامل بخادم Node.js:
- **العميل:** HTML/CSS/JS خالص + Canvas + WebSocket
- **الخادم:** Express + `better-sqlite3` + MongoDB (اختياري)
- **الرندر:** Isometric 2.5D + Sprite Factory + Canvas
- **التخزين:** SQLite (دائم) + localStorage (نسخ احتياطي)
- **الشبكات:** WebSocket لـ PvP + غارات الـ Battle Royale

## ✅ آخر التعديلات المنجزة (الجلسة الأخيرة)

### 📂 النظام الهيكلي للمشروع

```
الـ Hub (🏪 مركز التطوير)
├── 💪 ساحة الجيش     ← ترقية الجنود + تطوير التدريب + إجمالي القوة
├── 🗡️ الأسلحة        ← شراء/تجهيز/ترقية نجمية (1⭐-5⭐) — 6 أسلحة
├── 📜 المعرفة        ← شجرة الترقيات (4 مسارات: جيش، معرفة، دفاع، تجارة)
├── 🎁 المكافآت       ← يوميات + إنجازات
└── 🛒 المتجر         ← بيع/شراء موارد (قطع أثرية، مخطوطات، مطارق، صرف عملات)
```

### 🔗 التوصيلات الجديدة (تمت في هذه الجلسة)

| النظام | مربوط بـ | التفاصيل |
|--------|----------|----------|
| 🗡️ الأسلحة | أيقونة الأسلحة 🗡️ | شاشة مستقلة مع شراء/تجهيز/ترقية |
| 💪 ساحة الجيش | أيقونة ساحة الجيش 💪 | ترقية الجنود والتدريب |
| 📜 شجرة المعرفة (مسار المعرفة) | 📖 القصة (knowledgeLevel) | 🔒 مقفل حتى تصل معرفة القصة للمستوى المطلوب |
| 📜 شجرة المعرفة (مسار الجيش) | 💪 GameArmy.unitLevel | 🔒 مقفل حتى يصل مستوى الجنود للمطلوب |
| 🛒 المتجر | موارد نادرة فقط | لا يحتوي أسلحة أو ترقية جنود |
