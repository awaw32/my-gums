## 🏗️ حالة المشروع الحالية (Baseline)

### ما هو موجود:
- لعبة استراتيجية صحراوية متعددة اللاعبين (MMO RTS)
- Frontend: Vanilla JS + Vite + CSS مخصص + Canvas 2D/2.5D Isometric
- Backend: Node.js + Express 5 + WebSocket (ws) + MongoDB + SQLite
- PWA مع Service Worker
- دعم كامل للغة العربية (RTL)
- توافق مع الجوال (Mobile-first responsive)

### الإصدار الحالي: `1.0.0`

1. اقرأ هذا الملف أولاً
2. افهم الأنظمة المرتبطة — كثير من الملفات تعتمد على بعضها
3. اختبر بعد كل تغيير صغير

### لإضافة ميزة جديدة:
1. حدد الملفات التي ستتأثر (استخدم `code-searcher`)
2. أضف التغيير
3. اختبر: `npm test`
4. شغّل اللعبة محلياً: `npm run dev`
5. اختبر يدوياً
6. حدث هذا الملف

### ملاحظات مهمة:
- اللعبة تستخدم **ثلاث طرق حفظ**: localStorage, API/HTTP, WebSocket
- أي تغيير في نظام الحفظ يؤثر على الثلاثة
- `GameStore` هو Event Bus — أي تغيير في الحالة العالمية يمر عبره
- `_markDirty` و `saveToDB()` مسؤولان عن حفظ التغييرات
- CSP Header في `server.js` — أي مصدر خارجي جديد يجب إضافته هناك
- ملف `settings.json` هو إعدادات VS Code **وليس** إعدادات اللعبة
